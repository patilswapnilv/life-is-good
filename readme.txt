Life Is Good
============
**Theme Name:** Life Is Good
**Theme URI:** http://www.github.com/patilswapnilv/life-is-good
**Author:** Swapnil V. Patil
**Author URI:** http://www.github.com/patilswapnilv/

Description
-----------
Life Is Good is a sleek personal blog theme based on the awesome TwentyThirteen theme from WordPress 

###Notes
* For best results, featured images should be at least 1000px in width.
* Galleries in a post with gallery post format are automatically displayed as a slider. For best results, these images should be at least 1000px in width.
